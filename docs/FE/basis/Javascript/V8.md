#### V8 执行代码

> V8 的代码执行是即时编译(Just-in-time compilation)，分为解释执行和编译执行，解释执行就是将源码转译为 V8 能够读懂的语言，编译执行

```
graph LR
(解释执行)
A[源代码] -->B[解析器]--> C[中间代码AST]--> D[解释器]--> E[执行]

```

```
graph LR
(编译执行)
A[源代码] -->B[解析器]--> C[中间代码AST]--> D[编译器]--> E[编译成机器代码]-->F[CPU执行]

```
